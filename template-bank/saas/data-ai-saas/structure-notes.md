# Structure Notes: Data & AI SaaS Starter

This document breaks down the structural logic and section-by-section sequencing for the intelligence-driven variant.

## 🧱 The Intelligence Rhythm (The "Data-to-Insight" Funnel)

Unlike the general SaaS backbone, this variant focuses on **Visibility** and **Predictive Power**.

### 1. Split-Impact Hero
- **Design:** Left-aligned text + Right-aligned rich dashboard preview.
- **Why:** Immediate visual proof that the tool provides data visualization and reporting.

### 2. Outcome Summary Bar
- **Design:** 4-column metric row (Outcome, Accuracy, Scale, context).
- **Why:** Converts trust into "quantified value" early in the page flow.

### 3. Horizontal Insight Flow
- **Layout:** 4-step horizontal process (Aggregation → Cleaning → Analysis → Insight).
- **Design:** Geometric connectors with icon-led value props.
- **Why:** Explains the "Black Box" of AI in a simple, understandable way.

### 4. Panel-Based Intelligence Feature Grid
- **Design:** High-density UI panels (`col-span-2` for key features).
- **Why:** Mimics the feel of a real analytical workspace or dashboard.

### 5. Deep-Dive Analytics Focus
- **Layout:** Vertical text focus with a large, interactive-style visual mockup.
- **Why:** Allows for detailed product storytelling focused on high-value analytics features.

### 6. Focused Conversion Section
- **Visuals:** Darker, high-contrast container with emerald/mint accents.
- **Action:** Single Primary CTA: "Unlock Insights" or "Connect Data Source."
