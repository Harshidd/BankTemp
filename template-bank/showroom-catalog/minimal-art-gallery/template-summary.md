# Minimal Art Gallery

## Overview
A minimal, elegant, exhibition-led art gallery website template. Built to emulate the calm, spacious visual rhythm of institutions like Esther Schipper and David Zwirner.

## Core Features
- **Exhibition-First Hierarchy:** Home and Exhibition pages place massive emphasis on the current visual dialogue before any artist roster.
- **Artists & Works Showcase:** Highly structured and clean roster tables/cards with restrained metadata (title, medium, size, year).
- **Extreme White Space:** Relies on generous spacing and minimal lines to drive focus entirely onto the art pieces.
- **WordPress-Friendly Structure:** Grounded components make it easy to adapt for static or headless CMS engines.
