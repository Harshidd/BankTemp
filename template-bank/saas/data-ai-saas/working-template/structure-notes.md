# Data & AI SaaS Structural Architecture

## Layout System
- **Framework:** Next.js 16 (App Router + React 19)
- **Styling:** Tailwind CSS v4 (Slate-950 base)
- **Routing:** State-based internal router in `page.tsx`.

## Page Components
- `pages/Home.tsx`: Dashboard-driven storytelling.
- `pages/Features.tsx`: Technical capability matrix.
- `pages/Solutions.tsx`: Sectoral application grids.
- `pages/Pricing.tsx`: Tiered intelligence packages.
- `pages/Contact.tsx`: Inquiry protocol hub.
- `pages/FAQ.tsx`: Technical help portal.

## Global Navigation Hubs
- `components/Navbar.tsx`: Fixed, backdrop-blur with state switching.
- `components/Footer.tsx`: Industrial link directory with system status.

## Shared Data Accents
- Unified background grid lines.
- Global indigo/emerald blur overlays.
- Dashboard-inspired paneling system.

## Performance Logic
- Zero manual refresh simulation via ML-anomaly mockups.
- High-velocity emerald interaction states.
- Sub-ms visual clarity for dense data grids.
