# Structure Notes

## Architectural Layout
- Single-page application (SPA) router built inside Next.js.
- Dynamically updates local view state via the `onPageChange` callback and routes smooth scrolling to top.
- Navigation links map perfectly between the Navbar, Footer, and in-page CTA buttons.

## Visual Design Details
- Warm, neutral, high-contrast color scheme tailored for a calm, editorial breathing room.
- Clear typographic structure using custom font-serif and font-sans tags.
- Flexible modular cards for easy expansion or future database mapping.
