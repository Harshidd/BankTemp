# Analysis: Modern B2B SaaS (V2 Multi-Page)

## Reference Infusion Logic (Structural Extractions)

This template serves as a **High-Velocity SaaS Hub**, leveraging the conversion and trust strengths of two primary references.

### 1. Minimalist Execution (Primary Influence: Linear)
- **Extraction:** The use of sub-millisecond sync architecture as a core value proposition.
- **Extraction:** Monochrome-driven typography with subtle monochrome gradients for depth.
- **Logic:** The "Product" first hero flow on the Home page is the primary conversion path for high-intent technical users.

### 2. Strategic Narrative (Secondary Influence: Figma)
- **Extraction:** The "Philosophy-driven About Hero" focused on "fragmentation of clarity."
- **Extraction:** Multi-page expansion for "Features" and "FAQ" to centralize research beyond the hero page.
- **Logic:** This template expands on the single-page landing logic of Linear to support the deeper Trust/Proof requirements seen in more established collaborative platforms like Figma.

### 3. Pricing Tiering (B2B SaaS Benchmarks)
- **Extraction:** The dedicated Pricing page is a critical structural pattern for B2B SaaS. It allows for future expansion of feature-rich comparison grids.
- **Logic:** 3-column pricing system (Starter, Premium, Enterprise) provides a clear growth path for teams.

### 4. Expansion Summary (The MRKDEV Approach)
- **Problem:** Many SaaS starters are either too flashy (consumer-focused) or too simple (single-page).
- **Solution:** This **MRKDEV Starter** provides a 6-page professional system that is **Niche-Agnostic**. It can be rebranded in < 30 minutes for any SaaS category (DevTools, HR, CRM, etc.) while maintaining high-velocity product storytelling.
- **Visual:** Maintains a premium, modern feel (Glassmorphism in the Navbar, high-contrast Dark-Light transitions, 3D shadows) that differentiates it from older, cluttered B2B SaaS websites.

## Excluded Branding (IP Protection)
- **Naming:** Does not use "Linear" or "Figma".
- **Visuals:** Uses generic, professional-grade Lucide icons instead of original branding icons from the references.
- **Copy:** All text is rewritten as reusable placeholders with specific localization paths (Inquiry Purpose, ZIP checks).
